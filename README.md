# ssb-desktop-notify

Desktop notifications for ssb/patchwork. Generates notifications on mentions,
digs, private messages, and follows.

Currently supports only freedesktop.org notifications.

## Install

```
npm install -g ssb-desktop-notify
```

## Usage

```
ssb-desktop-notify &
```
